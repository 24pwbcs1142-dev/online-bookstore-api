# Online Bookstore Management API

## Description
A RESTful API built using Node.js, Express, and MongoDB for managing an online bookstore.

## Features
- Add, update, delete, and view books
- Search by author and genre
- Pagination support
- Proper error handling

## API Endpoints
- GET /api/books
- GET /api/books/:id
- POST /api/books
- PUT /api/books/:id
- DELETE /api/books/:id

## Run Locally
1. Clone repository
2. Install dependencies: npm install
3. Create .env file
4. Run server: npm run dev
